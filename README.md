AeroChef AI Assistant
AeroChef AI is an intelligent, real-time culinary production assistant designed to streamline kitchen operations and safety protocols. Integrated directly into the AeroChef ecosystem, this AI-powered chatbot provides on-demand assistance for flight catering management, real-time batch tracking, and safety compliance checks.

🚀 Key Features
Real-time AI Agents: Access specialized bots for Chef Operations (menu management), Safety Compliance (protocol verification), and Ops Leadership (production scheduling).

Interactive Chat Interface: A low-friction, floating UI that allows staff to query production data without interrupting their current workflow.

Smart Production Tracking: Instantly visualize batch status, flight IDs, and completion targets via AI-generated, structured data reports.

Safety First: Immediate access to safety procedures to ensure all food production meets strict industry standards.

🛠 Tech Stack
Frontend: Blazor WebAssembly / Server with responsive Bootstrap CSS.

Backend: ASP.NET Core Web API.

AI Engine: Custom-built Agentic Framework (ChefBot, SafetyBot, OpsLead).

Communication: Asynchronous HttpClient integration with real-time UI state management.

💡 How It Works
AeroChef AI utilizes a modular agent architecture. Each agent is specialized for a specific domain, allowing the user to get precise, role-specific answers. The frontend features a "glassmorphism" styled chat interface that stays active across different modules of the application, ensuring help is always just one click away.
